# KinPlay Prompts

## If Toys Could Talk
- ID: `if_toys_could_talk`
- Type: prompt
- Modes: quick_play, pick_a_game
- Age: 2–8
- Duration: 5 min
- Energy: calm
- Summary: Imagine what a favorite toy would say.
- Materials: none
- Safety tags: quiet, no_materials, sibling_friendly
- Prompt: Pick a toy nearby. If it could talk for one minute, what would it ask your family?
- Follow-ups: What voice would it use? / What adventure would it want next?
